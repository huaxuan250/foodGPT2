# foodGPT2
A simple recipe answering machine that extends from my foodGPT: https://github.com/huaxuan250/foodGPT

It allows for 'vegie', 'meat', 'healthy', 'unhealthy', 'AMERICAN', 'double' and 'half' keyword transformation.
